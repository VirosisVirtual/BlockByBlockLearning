Mods

Mods are an optional addition to Minecraft that changes aspects of the game while adding its own features. The mods listed below are HIGHLY recommended to aid and speed up the building process.

WorldEdit is a straightforward map editor mod that has features which can quickly be picked up while being fast and easy to learn its core basics.

Axiom is a more advanced Minecraft editing mod that mimics the capabilities of a 3D modeling program, with numerous tools for selecting, drawing, and shaping the world.
